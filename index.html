Welcome to SWGRealms

SWGRealms offers an easy and affordable way to launch your own Star Wars Galaxies Emulator server based on the open source SWGEmu Code

Enjoy full control over settings and configuration with our user-friendly management interface.

Features

Easy-to-use Control Panel
Single Sign-on Across All Servers
Your Choice: Invite Only or Open to All Community Members
VPN and Bot Detection
Transaction Review Tooling
Account Wide Bans, IP Bans
Customizable Settings, Including Lua Script
Server Monitoring and Alerting
Automated Snapshot Backups
Advanced Plans Include Git Repo Automation
