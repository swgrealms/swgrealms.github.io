Self-Service Solutions

Before contacting support, try these common fixes:

  1. Clear your browser cache and cookies
  2. Try a different browser or incognito mode
  3. Check your email spam folder for verification messages
  4. Verify you're using the correct email address
  5. Disable browser extensions that might interfere with authentication
